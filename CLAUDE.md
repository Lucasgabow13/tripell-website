## DESIGN RULES

**Match everything on the inspiration site:**
- Color palette — extract exact hex codes from the site
- Font families, weights, and sizing scale
- Letter spacing, line height, and text treatment
- Section layouts, spacing, and padding proportions
- Navigation structure, style, and sticky scroll behavior
- Hero section layout and visual treatment
- Button styles, border radius, and hover states
- Card component styles and shadows
- Dividers, borders, and decorative elements
- Animations and scroll transitions
- Footer layout and structure
- Mobile and tablet breakpoints — match the responsive behavior of the inspiration site

**Do not:**
- Invent design elements that don't exist on the inspiration site
- Use a different color, font, or spacing that isn't present in the reference
- Add components the inspiration site doesn't have
- Use stock photos — placeholder blocks only where images are needed
